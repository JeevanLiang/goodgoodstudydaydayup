### Q1 判断质数

大于1的数，如果除了1和它本身，它不能再被其它正整数整除，那么我们说它是一个质数。判断一个数是不是质数，写个程序进行判断

##### 输入格式
输入包括一行，为一个整数 N(1<N\leq 1000)N(1<N≤1000)，给出你让你判断的数字。

##### 输出格式
输出包括一行，如果晓萌给出的整数N为质数，那么输出YES；如果N不是质数，那么输出NO

##### 样例输入
3

##### 样例输出
YES
